# DVD-Screen.JS
Link this js file to create a simple dvd screensaver.
#### File link: https://mrpotato-04.nl/public-js-modules/dvd-screen.js
#### Usage:
1.  link js script.
2.  create element with the id dvd-box
3.  specify width and height
